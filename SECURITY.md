# Security policy

## Supported versions

Only the latest tagged release on [GitHub Releases](https://github.com/joenb33/spectraglyph/releases) is supported with security fixes when applicable.

## Reporting a vulnerability

If you believe you have found a security issue in SpectraGlyph, please **do not** open a public issue.

Instead, contact the maintainer privately:

- **GitHub:** [@joenb33](https://github.com/joenb33) (use **Security advisories** on the repository if available, or direct message if you have a channel).

Include a short description, steps to reproduce, and affected version or commit when possible.
